maple的repo

标签（空格分隔）： repo git vcs maple

repo init 初始化一个repo库，下载清单文件

repo sync 根据清单文件同步代码

-h 帮助文档
-f 失败继续同步
-d 会将HEAD强制指向manifest的库，而忽略本地的改动

maple代码与裤上代码保持同步：

repo forall -c ‘git status；pwd’
